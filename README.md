# 📚 Student Record Management System (Python)

A simple yet practical **command-line based Student Record Management System** built using Python. This project allows users to manage student data efficiently with features like adding records, viewing details, saving data to files, loading existing records, and maintaining activity logs.

---

## 🚀 Features

* ➕ Add student details (Name, Age, Grade)
* 📋 View all student records in a structured format
* 💾 Save records to a file (`students.txt`)
* 📂 Load existing records from file
* 📝 Maintain logs of all actions with timestamps (`log.txt`)
* ⚙️ Menu-driven interface for easy interaction

---

## 🛠️ Technologies Used

* **Python 3**
* File Handling (Read/Write)
* Dictionaries (for data storage)
* Functions (modular programming)
* Datetime module (for logging)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/arshadMLdev/student-record-system.git
```

2. Navigate to the project folder:

```bash
cd student-record-system
```

3. Run the program:

```bash
python main.py
```

---

## 🧠 How It Works

* The program stores student data in a Python dictionary.
* Users interact through a menu-driven CLI.
* Data can be saved into a text file and reloaded when needed.
* Every major action is recorded in a log file with timestamps.

---

## ⚠️ Notes

* Ensure Python is installed on your system.
* The files `students.txt` and `log.txt` will be created automatically when needed.
* Avoid manually modifying the data file to prevent format issues.
